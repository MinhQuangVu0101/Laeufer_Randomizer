# Laeufer-Randomizer

Volleyball Team Generator — Erstellt faire Teams basierend auf Spielerpositionen mit einem Backtracking-Algorithmus.

## Features

- Spieler nach Position eingeben (Aussen, Mitte, Zuspieler, Libero, Diagonal)
- Spieler koennen mehrere Positionen haben
- Intelligente Teamverteilung mit Backtracking-Algorithmus
- Teams ohne Libero moeglich (pro Team konfigurierbar)
- Auswechselbank bei zu vielen Spielern
- Kader speichern und laden (localStorage)
- Teams als Text kopieren (fuer WhatsApp etc.)
- Letzte 3 Generierungen als History
- Dark Mode Design, mobilfreundlich

## Nutzung

1. Oeffne `index.html` im Browser (oder besuche die [GitHub Pages Version](https://minhquangvu0101.github.io/Laeufer_Randomizer/))
2. Gib Spieler pro Position ein (ein Name pro Zeile)
3. Klicke "Teams generieren"
4. Optional: Kader speichern fuer spaeter

## Lokal starten

```
git clone https://github.com/MinhQuangVu0101/Laeufer_Randomizer.git
cd Laeufer_Randomizer
# Einfach index.html im Browser oeffnen
```

Keine Dependencies, kein Build-Step.
